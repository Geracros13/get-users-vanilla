# 👥 User Directory Project

A dynamic and responsive user directory that consumes data from the **DummyJSON API**. This project implements core features of a modern data management application, such as real-time search, alphabetical sorting, and pagination.

## 🚀 Features

- **Dynamic Loading:** Asynchronous data fetching from an external endpoint using the Fetch API.
- **Real-Time Search:** Instant filtering by name or username (case-insensitive).
- **Alphabetical Sorting (A-Z):** Reorganize the current list with a single click.
- **Responsive Design:** Fully optimized for mobile and desktop using CSS Grid and Flexbox.
- **Accessibility & SEO:** Built with HTML5 semantic landmarks and descriptive meta tags.

## 🛠️ Tech Stack

- **HTML5:** Semantic markup and Accessibility (Landmarks).
- **CSS3:** Custom properties, Grid Layout, Flexbox, and Media Queries.
- **JavaScript (ES6+):** Fetch API, Async/Await, DOM manipulation, and Array methods (`filter`, `sort`, `map`).
- **DummyJSON API:** Data source for prototyping.
